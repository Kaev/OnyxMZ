# OnyxMZ
The goal of OnyxMZ is to offer basic MMO gameplay for RPG Maker MZ projects.

## First Steps
- Install [NodeJS](httos://nodejs.org)
- Download the latest release (or repository)
- Copy all files into your project folder
- Open a console and navigate to your project folder
- Run `npm install` in the project folder and inside the server folder
- Run `node server/server` to run the server
- Activate all available OnyxMZ plugins in the plugin manager of the RPG Maker MZ
- (optional) Configure server IP and port on OnyxMZ-Websocket plugin parameters
